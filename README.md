# AI-Powered CSV Importer for GrowEasy

This repository contains a full-stack hiring assignment project for importing CSV lead files and normalizing them into the GrowEasy CRM format with AI.

## Project Status

Phase 1 is complete: the repository has a minimal Next.js frontend and Express backend health endpoint.

Later phases will add CSV preview, AI-powered mapping, import results, CSV download, deployment notes, and the full README content required by the assignment.

## Folder Structure

```text
/frontend
/backend
README.md
```

## Local Development

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

The backend health endpoint will be available at:

```text
http://localhost:5000/api/health
```

The frontend will be available at:

```text
http://localhost:3000
```
